## Featured Projects

### Mini CMS
A lightweight multilingual Laravel CMS for small websites, portfolios, business websites, blogs, and shared-hosting environments.

**Focus:** Laravel, Filament, Livewire, Tailwind CSS, SQLite, multilingual content, RTL support, installer workflow, and production deployment.

[![Mini CMS](https://img.shields.io/badge/View%20Repository-mini--cms-7C3AED?style=for-the-badge&logo=github)](https://github.com/mtalavi/mini-cms)

---

### DPVHUB
A volunteer and event management platform with DP codes, QR cards, event registration, attendance control, admin roles, exports, and audit logs.

**Focus:** PHP, MySQL, QR workflows, admin operations, event check-in/check-out, and lightweight deployment.

[![DPVHUB](https://img.shields.io/badge/View%20Repository-dpv-0EA5E9?style=for-the-badge&logo=github)](https://github.com/mtalavi/dpv)

---

### VoIP Phone Dialer
A mobile-first WebRTC voice app with user registration, 6-digit phone numbers, call history, ICE configuration, reconnect handling, and deployment notes.

**Focus:** Node.js, Express, Socket.IO, WebRTC, TURN configuration, and Docker deployment.

[![VoIP](https://img.shields.io/badge/View%20Repository-voip-F97316?style=for-the-badge&logo=github)](https://github.com/mtalavi/voip)

---

### Norino
A modern web app concept around user/expert workflows, OTP login, dashboards, request/response handling, subscriptions, and admin controls.

**Focus:** Next.js, React, Prisma, dashboards, user roles, OTP workflow, and notification flows.

[![Norino](https://img.shields.io/badge/View%20Repository-norino-EC4899?style=for-the-badge&logo=github)](https://github.com/mtalavi/norino)
